# chainer-waifu2x

Yet another implementation of [waifu2x](https://github.com/nagadomi/waifu2x) using [chainer](http://chainer.org).

# Models

The file `anime_style_art_rgb_scale2.0x_model.json` is imported from [the original waifu2x repository](https://github.com/nagadomi/waifu2x).
